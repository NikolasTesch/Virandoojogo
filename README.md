# Virando o Jogo ⚽🏀

> **Projeto esportivo e cultural da Prefeitura de Teixeira de Freitas, BA.**

O **Virando o Jogo** é uma iniciativa gratuita voltada para jovens de **13 a 20 anos**, oferecendo treinamento esportivo de alto nível, disciplina e desenvolvimento pessoal. Comandado pelo técnico **Bruno Gomes** (treinador da Seleção Municipal), o projeto visa transformar o potencial de jovens talentos em poder real.

## 🚀 Sobre o Projeto

Este repositório contém o código-fonte da Landing Page oficial do projeto. A página foi desenvolvida com foco em:
- **Alta Performance**: Estrutura leve e otimizada (HTML5/CSS3).
- **Design Premium**: Interface moderna, responsiva e com foco em usabilidade.
- **Micro-animações**: Experiência dinâmica para o usuário.
- **Acessibilidade & SEO**: Seguindo as melhores práticas para garantir que o projeto chegue a quem precisa.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Estilos personalizados, Flexbox, Grid e animações.
- **JavaScript**: Interatividade e menu mobile.
- **Google Fonts**: Tipografia moderna (Bebas Neue, Barlow).

## 📂 Estrutura de Arquivos

```text
/
```

## 💻 Como Executar

Para visualizar o projeto localmente:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/virandoojogo.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.
